This game is in active development, and needs some finishing touches to fit the official Tetris Guidelines. However, the core gameplay element should be completed.


Controls:

Up: Rotate Clockwise
Down: Soft Drop
Left: Move Piece Left
Right: Move Piece Right
Z: Rotate Counter Clockwise
A: Rotate 180
Space: Hard Drop
C: Hold Piece
P: Pause Game

For those interested, here are the things.

* Scoring Guidelines
* * PC detection
* * TSpin Detection
* * B2B detection
* * Combo Detection (Done, but not shown in UI)

* SRS kick table (Making it FEEL like tetris, and allowing for T-spins, the most glaring problem)
* DAS and ARR (control conveniences)
* sound (planning on using the minecraft tetr.io soundpack here)
* some better UI (text specifically)
* touch controls (like jstris)
